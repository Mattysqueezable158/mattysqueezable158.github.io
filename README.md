# Hacıveyiszade Otomatik Kapı Açma Projesi

Bu proje, bir **ESP32-S3** mikrokontrolcü geliştirme kartı ve **Supabase** bulut veritabanını kullanarak, **Next.js** ile geliştirilmiş şık bir web arayüzü üzerinden kapı açma mekanizmasını kontrol etmenizi sağlar.

---

## Sistem Mimarisi ve Çalışma Mantığı

1. **Next.js Portal (Frontend)**: Giriş yapmış yetkili kullanıcı "KAPIYI AÇ" butonuna bastığında Supabase veritabanındaki `door_requests` tablosuna `status = 'pending'` olan yeni bir satır ekler.
2. **Supabase (Veritabanı)**: Veriyi güvenli bir şekilde saklar. RLS (Row Level Security) kuralları sayesinde yetkisiz erişimleri engeller.
3. **ESP32-S3 (Donanım)**: Her 1.5 saniyede bir Supabase REST API'sini sorgulayarak bekleyen (`pending`) bir istek olup olmadığına bakar.
4. **Tetikleme (Pin 11)**: İstek bulunduğunda, ESP32-S3 Pin 11 çıkışını 2 saniyeliğine aktif ederek röleyi tetikler ve kapıyı açar. Ardından veritabanındaki isteği `completed` durumuna getirir. Web arayüzü bu durum değişikliğini anlık algılayarak kullanıcıya geri bildirim verir.

## 1. Adım: Supabase Veritabanı Kurulumu

Supabase yönetim panelinizde **SQL Editor** bölümüne giderek aşağıdaki SQL betiğini çalıştırın. Bu betik gerekli tabloyu oluşturacak ve RLS güvenlik kurallarını tanımlayacaktır:

```sql
-- Kapı istekleri tablosunun oluşturulması
create table door_requests (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  user_id uuid references auth.users(id) on delete set null,
  status text not null default 'pending', -- 'pending', 'processing', 'completed', 'failed'
  processed_at timestamp with time zone,
  requested_by_email text
);

-- RLS (Row Level Security) Aktifleştirme
alter table door_requests enable row level security;

-- Güvenlik Politikaları (Policies)
-- 1. Giriş yapmış kullanıcılar kapı açma isteği ekleyebilir
create policy "Kullanıcılar kapı açma isteği ekleyebilir" 
on door_requests for insert 
to authenticated 
with check (true);

-- 2. Giriş yapmış kullanıcılar ve ESP32 tüm istekleri okuyabilir
create policy "Herkes istekleri okuyabilir" 
on door_requests for select 
using (true);

-- 3. ESP32 istek durumunu güncelleyebilir (status = 'completed')
create policy "İstek durumu güncellenebilir" 
on door_requests for update 
using (true);
```

### Yetkili Kullanıcı Ekleme
Web arayüzünden giriş yapabilmek için Supabase panelinde **Authentication > Users** kısmından yeni bir kullanıcı (e-posta ve şifre ile) ekleyin.

## 2. Adım: Next.js Uygulaması Kurulumu

### Ortam Değişkenleri
Proje kök dizininde bulunan `.env.local` dosyasını açın ve Supabase bilgilerinizle güncelleyin:

```env
NEXT_PUBLIC_SUPABASE_URL=https://<PROJE_ID>.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=<SUPABASE_ANON_KEY>
```

### Uygulamayı Çalıştırma
Bağımlılıkları yükleyin ve yerel geliştirme sunucusunu başlatın:

```bash
# Geliştirme modunda başlat
npm run dev
```

Tarayıcınızda `http://localhost:3000` adresine giderek, eklediğiniz kullanıcı bilgileri ile giriş yapabilirsiniz.

## 3. Adım: ESP32-S3 Donanım Bağlantısı

ESP32-S3 kartı ile röle kartı arasındaki bağlantıları şu şekilde yapın:

| ESP32-S3 Pini | Röle Modülü Pini | Açıklama |
| :--- | :--- | :--- |
| **GND** | **GND** | Ortak Toprak |
| **5V / 3.3V** | **VCC** | Besleme Gerilimi (Röle voltajına göre) |
| **GPIO 11** | **IN / SIG** | Röle Tetik Sinyali |

### Kapı Otomatiği Bağlantısı:
Rölenin **COM** (Common - Ortak) ve **NO** (Normally Open - Normalde Açık) klemenslerini, kapı otomatiğinizin duvar butonunun (veya interkom ünitesindeki anahtarın) iki ucuna paralel olarak bağlayın. Bu sayede röle kapandığında buton fiziksel olarak basılmış gibi kapı açılacaktır.

## 4. Adım: ESP32-S3 Yazılım Yükleme (Arduino IDE)

1. [firmware/firmware.ino](file:///c:/Users/ferhatozcelik/Desktop/door/haciveyiszade.github.io/firmware/firmware.ino) dosyasını Arduino IDE ile açın.
2. Kodun başındaki şu değişkenleri kendi ağınıza ve Supabase projenize göre düzenleyin:
   ```cpp
   const char* ssid = "YOUR_WIFI_SSID";          // Wi-Fi Adı
   const char* password = "YOUR_WIFI_PASSWORD";  // Wi-Fi Şifresi
   const char* supabaseUrl = "https://your-project-ref.supabase.co"; // Supabase URL
   const char* supabaseAnonKey = "your-supabase-anon-key";         // Anon Key
   ```
3. Arduino IDE'de **Tools > Board** menüsünden **ESP32S3 Dev Module** (veya kullandığınız tam ESP32-S3 modelini) seçin.
4. Kartı bilgisayarınıza bağlayın, doğru COM portunu seçin ve **Upload** butonuna basarak kodu yükleyin.
5. Seri Port ekranını (Baud rate: 112500) açarak Wi-Fi bağlantısını ve veritabanı sorgularını takip edebilirsiniz.
