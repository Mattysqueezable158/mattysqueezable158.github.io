## 🎮 Using the manager
Once you install the tool, you can start customizing your game.

### Starting the program
Find the icon on your desktop or in your start menu. Click it to open the main window. The program will scan your computer for your game installation. If it fails to find the game, use the settings menu to point the tool to your game folder.

### Adding your first mod
1. Locate your mod files. These files usually end in ".fifamod" or ".fbmod".
2. Open the manager window.
3. Drag your mod file into the main list area.
4. The tool adds the file to your mod list automatically.

### Controlling load order
Some mods change the same items in the game. When this happens, the tool displays a warning. You can resolve this by moving mods up or down the list. The tool loads mods at the bottom of the list over mods at the top. Use the mouse to drag and drop rows to change the priority.

### Launching the game
Click the green "Launch" button inside the mod manager. This button applies your selections and starts the game with your chosen mods visible. If you leave the manager and start the game through EA App or Steam directly, your mods will not load.

## 🛠 Troubleshooting common issues
Sometimes mods do not work as expected. Follow these steps to fix common problems.

### Game crashes on start
If the game closes instantly, one of your mods might be outdated. Disable all mods by unchecking the boxes next to their names. Launch the game again. If the game opens, one of your active mods is the cause. Enable them one by one to find the faulty file.

### Mods do not show in-game
Check the settings menu to ensure your game path is correct. Verify that the "Enable Mods" toggle is in the "On" position. Refresh the mod list to make sure the manager recognizes your files.

### Clearing the cache
The program creates temporary files to speed up the loading process. If the game acts strange, clear these files. Go to "Settings" then "Clear Cache." The program will restart and rebuild the data during the next launch.

## 🛡 Security and safety
Customizing game files carries some risk. Always create a backup of your save files before you install new mods. This tool does not change your original game files directly. It creates a local copy and injects the changes during the start sequence. If you remove the manager or disable all mods, your game returns to its original state. Use only mod files from sources you trust to keep your computer stable.
